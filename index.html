Data Mapping

  1. Upload: CSV/Excel with population data
  2. Select: Year & Population columns
  3. Merge: Match data to shapefile locations
  4. Visualize: Per-year choropleth maps
  5. Export: PNG maps & Excel/CSV data

Smart Population Scale

  • Auto-detect: Reads your actual max value
  • < 1,000: Raw numbers (e.g. 450)
  • ≥ 1,000: K suffix (e.g. 12.5K)
  • ≥ 1,000,000: M suffix (e.g. 3.2M)
  • ≥ 1,000,000,000: B suffix (e.g. 1.4B)

Scale Modes

  • Global: One scale across all years
  • Per-Year: Each map scales to its year
  • Log scale: For skewed distributions
  • Equal intervals: Even class breaks
  • Quantile: Equal-count class breaks
